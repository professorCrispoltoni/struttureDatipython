# ES3: Temperature e Statistiche

**Dati:** `giorni = ("lun","mar","mer","gio")`

**Obiettivo:** Media, deviazione standard, giorni > media.

**Input esempio:**
```
inserisci temp giorno lun: 10
inserisci temp giorno mar: 15
...
```

**Output atteso:**
```
media 12.5
dev_standard 2.38
dev_standard_formula 2.38
giorni con temperatura sopra la media
mar
```

**Istruzioni:**
1. Crea lista temp da input
2. Calcola media manuale
3. Dev std manuale + con `statistics.stdev`
4. Stampa giorni > media

**Livello:** Tuple + cicli + statistiche
