# ES6: Gestione Palline Colorate

**Comandi:** "blu", "rossa", "del" (ultimo), "stop"

**Input esempio:**
```
Inserisci rossa o blu: blu
Inserisci rossa o bl: rossa
Inserisci rossa o bl: del
Inserisci rossa o bl: stop
```

**Output atteso:**
```
Sequenza palline: ['blu']
Conteggi: blu=1 rossa=0
```

**Istruzioni:**
1. Lista palline + dict conteggi
2. While True fino "stop"
3. "del" pop() e decrementa
4. Stampa finale

**Livello:** Liste + dict + while
