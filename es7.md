# ES7: Valutazione Voti Materie

**Materie:** ['Matematica', 'Filosofia', 'Geografia', 'Musica']

**Fasce:** A:[70-100], B:[60-69], ..., F:[0-29]

**Input esempio:**

```
Che voto hai preso in Matematica? 85
```

**Output:** `{'Matematica': 'A', 'Filosofia': 'C', ...}`

**Istruzioni:**
1. Dict fasce_giudizi
2. Per ogni materia: while per voto valido (0-100)
3. Trova fascia corrispondente
4. Stampa dict risultati

**Livello:** Dict annidati + validation input
