# Esercitazione Python - Gestione Squadra

## Obiettivo
Dato un dizionario che rappresenta una squadra (nome → età), scrivere un programma Python che esegua diverse operazioni sui dati.

## Dati di partenza

```python
squadra = {
    "Mario": 17,
    "Luca": 25,
    "Giulia": 19,
    "Sara": 12
}

Richieste

Scrivere un programma che:

- Stampi tutti i giocatori maggiorenni (età ≥ 18)
- Trovi il giocatore con età minima
- Trovi il giocatore con età massima
- Calcoli la media delle età

Output atteso
Giocatori maggiorenni:
Luca - 25
Giulia - 19

Età minima: 12 (Sara)
Età massima: 25 (Luca)
Media età: 18.25
